stbui - 前端集成解决方案

       stbui的目标是将你从繁琐的构建配置中解放出来，同时还省去每个项目都要安装一堆开发依赖的麻烦。

JS 文件的分析、合并、压缩、版本管理

CSS 文件的分析、合并、压缩、版本管理

HTML 文件分析、压缩、版本管理

入口模块的分析、分块构建、增量构建、合并、压缩、版本管理

静态资源的分析、压缩、版本管理

兼容 ie6 - ie11/chrome/firefox，以及各种手机浏览器

支持分块加载和异步加载

良好的构建工具配套支持

助你开始使用 stbui 的简易指南，木有高深内容，;)。

安装

npm installl -g stbui

使用命令创建项目

在当前目录创建为demo的项目例子
stbui project demo

启动项目服务

stbui server start

构建项目

stbui release

访问项目

http://127.0.0.1:8080

项目零开发依赖

省去项目内安装多个开发依赖的过程，从创建项目到启动开发环境花费时间不到十秒。

脚手架

内置脚手架功能，能快速创建包含 stbui 配置文件的项目。
